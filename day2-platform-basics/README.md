# Day 2 - Salesforce Platform Basics

---

# 1. What is Salesforce Platform?

Salesforce Platform is a cloud-based platform that allows businesses to build CRM applications using low-code and code-based development tools.

It provides:
- Objects
- Apps
- Automation
- Security
- Apex
- Lightning Components

---

# 2. Key Concepts

## What is an App?

An App is a collection of related tabs and objects designed for a business purpose.

Example:
- Sales App
- Service App

---

## What is an Object?

An Object is a database table used to store records.

Examples:
- Account
- Contact
- Student

---

## What is a Tab?

A Tab is a user interface element used to access objects and features.

---

# 3. Configuration vs Coding

## Configuration

Used when business requirements can be solved using built-in Salesforce tools.

Examples:
1. Validation Rules
2. Flow Automation

## Coding (Apex)

Used for complex business logic.

Examples:
1. API Integration
2. Advanced Scheduling Logic

# 5. CRM Concepts

Accounts, Contacts, and Opportunities are standard Salesforce objects used in CRM processes.

These objects are organized inside Apps to help users manage business operations efficiently.
